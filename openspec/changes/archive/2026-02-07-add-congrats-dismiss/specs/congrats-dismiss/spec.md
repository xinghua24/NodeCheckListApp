## ADDED Requirements

### Requirement: Dismiss congratulations overlay
The system SHALL allow users to dismiss the congratulations overlay either by clicking a close control on the card or by clicking the overlay background outside the card.

#### Scenario: Dismiss via close control
- **WHEN** the user clicks the close control on the congratulations card
- **THEN** the congratulations overlay is dismissed

#### Scenario: Dismiss via overlay background
- **WHEN** the user clicks the overlay background outside the congratulations card
- **THEN** the congratulations overlay is dismissed
